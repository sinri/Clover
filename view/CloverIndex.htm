Install

Clover is easy to install. Just download the toolkit codes and configure the interface page. Then add the Rewrite Code to the file .htaccess. 

	RewriteEngine On
	RewriteCond %{REQUEST_FILENAME} !-f
	RewriteCond %{REQUEST_FILENAME} !-d
	RewriteRule ^(.*)$ index.php [QSA,L]
	# For Apache2

Startup for Web Request

For web request, you need to generate the controllers, models and views in certain directories, which are defined in the Clover Class and could be changed dynamically.

Then you can open your browser to link to your site with url. The url rules are as following. The part of index.php might be neglected. 

	WEB_ROOT/CONTROLLER(/ACTION(/PARAM))

Startup for CLI

For CLI (designed for cronjob), the models and views are same to web request, but no controllers needed, instead commands are required.

Run script command as following. 

	php CloverCLI.php COMMAND ACTION --PARAM_NAME=PARAM_VALUE
